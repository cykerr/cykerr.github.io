【真题】金字塔最大路径
发表于 2020-05-04| 更新于 2020-05-06|计算机刷题
字数总计:1k|阅读时长: 3 分钟
| 阅读量:|评论数:

题目

在一个N层高的金字塔上，以金字塔顶为第一层，第i层有i个落点，每个落点有若干枚金币，在落点可以跳向左斜向下或向右斜向下的落点。若知道金字塔的层数N及每层的金币数量分布，请计算在本次游戏中可以获得的最多金币数量。

1. 输入

输入共有N + 1行(N ≤ 1024)，第一行为高度N，第二行至N + 1行 ，为该金字塔的金币数量分布。

2. 输出
输出金币数量。

3. 示例

解法

先说思路：

用一个二维数组D存放数字金字塔，D（i，j）表示第 i 行，第 j 列的数字，i，j 都从 0 开始计数。max_sum（i，j）表示从D（i，j）到底边的各条路径中，最大数字和。

从而，问题即求max_sum（0, 0），需要注意的是，从D（i，j）出发，下一步只能走D（i+1，j）或 D（i+1，j+1），直到 i == n，到达最底边，返回一个数值。

普通递归

时间复杂度：子问题有2n2^n2n个，而一次子问题的计算是比较两个节点大小然后执行一次加法操作，时间复杂度为O(1)O(1)O(1)，所以总体时间复杂度即O(2n)O(2^n)O(2n)。

空间复杂度：每个子问题需要的辅助空间是一个常数，递归深度是2n2^n2n，故空间复杂度为O(2n)O(2^n)O(2n)。

备忘递归

时间复杂度：因为不存在重复计算，只需要计算nnn个节点，所以总体时间复杂度即O(n)O(n)O(n)。

空间复杂度：每个子问题需要的辅助空间是一个常数，递归深度是nnn，故空间复杂度为O(n)O(n)O(n)。
